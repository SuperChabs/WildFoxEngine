# WildFoxEngine - Modern C++ Game Engine
A lightweight, modular 3D game engine built with modern C++ (C++23 modules), OpenGL, and Dear ImGui. 
Features an Entity Component System (ECS) architecture and a full-featured editor interface.

## Technology Stack
 C++23 - Modern C++ with module system
 OpenGL 3.3+ - Cross-platform graphics API
 GLFW - Window and input management
 GLAD - OpenGL loader
 GLM - Mathematics library
 EnTT - Fast and reliable ECS
 Dear ImGui - Immediate mode GUI
 Assimp - 3D model loading
 stb_image - Image loading
 nlohmann/json - JSON parsing
