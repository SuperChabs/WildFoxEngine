# WildFoxEngine - Modern C++ Game Engine
A lightweight, modular 3D game engine built with modern C++ (C++23 modules), OpenGL, and Dear ImGui. 
Features an Entity Component System (ECS) architecture and a full-featured editor interface.

## Technology Stack
1. C++23 - Modern C++ with module system
2. OpenGL 3.3+ - Cross-platform graphics API
3. GLFW - Window and input management
4. GLAD - OpenGL loader
5. GLM - Mathematics library
6. EnTT - Fast and reliable ECS
7. Dear ImGui - Immediate mode GUI
8. Assimp - 3D model loading
9. stb_image - Image loading
10. nlohmann/json - JSON parsing
