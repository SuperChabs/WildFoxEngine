\doxysubsection{Detailed Description}
Encapsulates all rendering functionality. 

Responsibilities:\+
\begin{DoxyItemize}
\item Initialize Open\+GL context
\item Create and manage \doxylink{classRenderer}{Renderer}
\item Manage resources (shaders, textures, materials, models)
\item Create \doxylink{classSkybox}{Skybox}
\item Setup rendering systems 
\end{DoxyItemize}
